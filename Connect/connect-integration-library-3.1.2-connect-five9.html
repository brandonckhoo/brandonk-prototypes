Before you start
Connecting Five9 requires admin access to your Five9 VCC. Estimated setup time: 30 minutes, or longer if you need to coordinate with your Five9 admin.
Inside Five9 VCC
✓ 1. Create a VCC API user with Read Config + Manage SIP Trunks permissions Docs ↗
✓ 2. Add Retell as a SIP trunk in Five9 VCC Admin → Telephony → SIP Trunks (use the SIP endpoint below) Docs ↗
3 3. Configure routing — assign campaigns, queues, and skills to the new trunk; set up dial plan Docs ↗
Inside Retell
4 4. Paste credentials below and validate the SIP + VCC API connection
5 5. Assign Five9 routing to phone numbers in the Phone Numbers section
6 6. Verify outbound compliance rules in Five9 VCC (TCPA, DNC, state windows)
Need help? Read the Five9 setup guide or contact our solutions team for a guided setup call.
