About
Channel routing is configured per agent in the Agent Builder. Once connected, any agent can send messages to the channel of their choice.
Written to Slack
Contact Caller name and phone number — referenced in template
Transcript link Signed URL to full transcript in Retell
Agent metadata Agent name, workspace, call duration, cost
Custom fields Any variables you reference in the message template
Read from Slack
channels Public and private channel list for the channel picker
users Workspace member directory for resolving @mentions
workspace.meta Workspace name and team ID for display
Retention. Retell does not store messages posted to Slack. Channel and user lists are cached for 15 minutes to populate pickers. Revoking access deletes the cache immediately.
