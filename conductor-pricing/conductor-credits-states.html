Conductor — Credits & Overage
  • Credit based freemium: 30 free credits per user per day, with a shared cap of 200 credits per workspace per day. 1 credit per turn, and there is no "buy credits" purchase.
  • Past the free limit: if the workspace has overage on, Conductor keeps working and bills $0.20 per credit up to the monthly cap. If overage is off, Conductor pauses until the daily reset.
  • Admin only: only workspace Admins can enable overage. Members are pointed to an Admin (see frame 4).
  • Enabling overage: happens on the Limits page via "Manage overage", not a modal.
  • Auto compression, no new-chat prompt: Conductor automatically compresses long chats to keep context and cost in check, so a user can stay in one chat indefinitely with no extra cost to Retell. We do not prompt users to start a new thread.
