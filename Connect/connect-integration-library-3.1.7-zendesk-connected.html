Agent tools
Functions your agents can call during conversations.
search_tickets(query)
Search Zendesk tickets by subject, requester, status, or tag. Returns up to 10 matching tickets.
get_ticket(ticket_id)
Fetch full ticket detail including conversation history, status, and custom fields.
create_ticket(subject, description, requester)
Open a new Zendesk ticket during the call — e.g., when a caller reports a new issue.
update_ticket(ticket_id, status, tags)
Update an existing ticket's status (open, pending, solved) or add tags for downstream routing.
add_ticket_comment(ticket_id, body, internal)
Post a comment to a ticket timeline. Use internal=true for private agent notes, false for customer-visible.
search_users(email_or_phone)
Look up a Zendesk user by email or phone number — returns contact, org, and any active tickets.
log_call_summary(ticket_id, transcript, outcome)
Auto-post a call summary with transcript link onto the ticket after the call ends.
Description
Connect Retell to Zendesk to let agents look up customer tickets, create and update tickets during calls, log call notes against contacts, and sync ticket context back into agent variables.
