Event types, availability, buffers, and booking limits are configured in Cal.com. Retell reads them at call time, so any change you make in Cal.com takes effect on the next call.

What Retell reads from your Cal.com account:

Event types
All event types on the connected account. The agent picks the right one at booking time based on the caller's intent.
Availability
Host calendars, working hours, buffers, and booking limits, exactly as configured in Cal.com.
Booking questions
Required fields on each event type. The agent will ask the caller for these during the call.
