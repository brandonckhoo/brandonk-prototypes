Billing Agent Filter
Feature specification — Filter Billing History by agent on the Billing tab
Use Case Description Screenshot
Filter Billing History for a specific agent
Billing Tab
Entry
User clicks on the Billing tab in the sidebar.
Flow
  1. 1User opens the Billing page.
  2. 2User navigates to the Billing History tab.
  3. 3The table displays all invoices across the workspace by default, with total amounts per billing period.
  4. 4User clicks the Filter button above the table.
  5. 5A dropdown appears with a searchable list of agents. User selects one or more agents and clicks Save.
  6. 6The active filter appears as a pill tag. All invoice row amounts update to reflect only the selected agent's usage.
  7. 7User expands a billing period row. The line-item breakdown (Voice LLM, Voice Engine, Telephony, etc.) and usage minutes all update to show only the filtered agent's contribution.
  8. 8User clicks × on the filter pill to clear the filter and restore full workspace totals.
1 — Default: full workspace billing history
2 — Filter dropdown: select agent(s)
3 — Filter active: row totals scoped to agent
4 — Expanded row: line-item breakdown for agent
View breakdown of branded calling and base carrier costs
Billing Tab
Entry
User selects an agent from the filter and expands the Telephony row in the invoice line-item breakdown.
Flow
  1. 1User opens the Billing page and selects one or more agents from the filter.
  2. 2User expands a billing period row and locates the Telephony line item in the breakdown.
  3. 3User clicks the expand toggle on the Telephony row.
  4. 4Nested rows appear showing US Telephony (per second) and Branded Calling as separate line items, each with a unit count, unit rate, and subtotal.
  5. 5User identifies which component is driving telephony cost for the selected agents.
5 — Telephony breakdown: base carrier vs Branded Calling
6 — 2 agents filtered: Branded Calling contribution
