Disconnect Zendesk?
This action will stop the integration for all agents using it.

Disconnecting will immediately stop Zendesk tools from working on live calls. Agents currently configured to use Zendesk will fail to create tickets, look up customers, or log call activity until the integration is reconnected.

Affected agents 3 agents
  • Support Triage AgentProduction
  • Returns & Refunds AgentProduction
  • Billing Escalation AgentStaging
  • OAuth token will be revoked with Zendesk.
  • Call history and historical sync records will be preserved.
  • You can reconnect Zendesk at any time from the Integrations page.
